# Smart Canteen Preorder System

## Description
A full-stack MERN application that allows students to preorder food from the college canteen, schedule pickup times, and track their orders.

## Features
- User Authentication
- Food Ordering
- Cart Management
- Order Tracking
- Admin Dashboard

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## Installation

### Backend
```bash
npm install
npm start
```

### Frontend
```bash
npm install
npm run dev
```
